First Installation

Each platform has slightly different steps, so choose the card that matches your device.

Windows / Linux

  1. Download the game `.zip` file and open it.
  2. Drag the folder inside the zip to a normal location such as your desktop, downloads folder, or games folder.
  3. Open the extracted folder and launch the file that matches your system:
  • 64-bit Windows: Stranded.exe
  • 32-bit Windows: Stranded-32.exe
  • 64-bit Linux: Stranded.sh
On Linux, if the file opens in a text editor instead of launching, right-click Stranded.sh and choose “Run as Program”. If needed, open a terminal in that folder and run chmod +x Stranded.sh && ./Stranded.sh.

Mac

  1. Download the game `.zip` file and open it.
  2. Drag the app inside the zip to your preferred folder, such as `Applications`.
  3. Double-click the app to launch it.
If macOS blocks the app, Control-click it, choose “Open”, then choose “Open” again in the warning prompt. If you still hit a permissions issue, move the app out of the zip first and try again.

Android

  1. Download the game `.apk` file, then open it.
  2. If Android shows a security warning, tap “Settings”.
  3. Enable installation from this source.
  4. Return to the installer and tap “Install”.

Updating Current Build

Updating is mostly the same as a first install, with one important Android warning.

Windows / Linux / Mac

  1. Download the newest build.
  2. Extract or move it just like you did the first time.
  3. Open the new version from its folder or app file.
Your saves are stored in a different folder, so replacing the old build will not remove them. You can safely delete the older game files after confirming the new version launches correctly.

One caution: loading a save made in a newer build with an older build can cause errors.

Android

Do not uninstall the previous app if you want to keep your saves. Uninstalling the old app can delete your save data.

Download the new `.apk` and install it over the existing version. That keeps your saves in place.
